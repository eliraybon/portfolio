EXO

Exo is sci-fi themed online store. Built with the MERN Stack + GraphQL & Apollo. Makes extensive use of several APIs (NASA, SWAPI) for database seeding. Inspired by Etsy.

Live - Github

ELECTRON ALIGNER

The Electron Aligner is an interactive step sequencer and drum machine. Chiptune sounds are synthesized from scrach using the Web Audio API and synchronized with the help of Tone.js. Using the looping sequencer, users can create their own beats live!

Live - Github
